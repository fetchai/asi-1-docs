# Chat completion example
A very simple example of how to get a chat completion with the `asi1-mini` model.

The **Chat Completion API** enables developers to interact with the `asi1-mini` model by sending messages and receiving AI-generated responses. Designed for natural language understanding and multistep reasoning, this API allows for contextual awareness and adaptive interactions, making it a powerful tool for autonomous [Agents](https://uagents.fetch.ai/docs/) and Web3 applications.

By sending a structured conversation history to the `chat/completions` endpoint, you can generate responses tailored to the given context. The API supports both standard and streaming responses, providing real-time interaction when needed. Parameters such as `temperature` allow for fine-tuned control over response variability, ensuring the model behaves as expected in different scenarios.

The following example demonstrates how to make a basic API request to ASI1 for chat completion.

## Code

```py copy filename="demo.py"

import requests
import json

url = "https://api.asi1.ai/v1/chat/completions"

payload = json.dumps({
  "model": "asi1-mini",
  "messages": [
    {
      "role": "user",
      "content": "Hi, tell me about giraffes"
    }
  ],
  "temperature": 0,
  "stream": True,
  "max_tokens": 0
})
headers = {
  'Content-Type': 'application/json',
  'Accept': 'application/json',
  'Authorization': 'Bearer your_api_key'
}

response = requests.request("POST", url, headers=headers, data=payload)

print(response.text)

```

You can run this by typing `python demo.py`. 

You may need to first install `requests` library to your environment for this to work; you can do that with: `pip install requests`.
